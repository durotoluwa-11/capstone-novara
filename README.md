# 🎓 Capstone Project - TaskApp Cloud Deployment

## Overview
TaskApp is a cloud-native task management application deployed on AWS using Kubernetes, Kops, and Terraform.

**Student**: Durotoluwa  
**Domain**: durotoluwa-tasked.com  
**Region**: us-east-1  

## Live URLs
- Frontend: https://taskapp.durotoluwa-tasked.com
- Backend API: https://api.durotoluwa-tasked.com

## Tech Stack
| Component | Technology |
|-----------|------------|
| Frontend | React (TypeScript) |
| Backend | Flask (Python) |
| Database | PostgreSQL |
| Cloud | AWS (us-east-1) |
| Kubernetes | Kops 1.28.5 |
| IaC | Terraform |
| Ingress | NGINX |
| SSL | cert-manager + Let's Encrypt |
| DNS | AWS Route53 |
